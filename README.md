## 🧩 О проекте

Мой личный сайт-портфолио, где я собрал свой опыт в дизайне и информацию о себе.

Основная задача - показать проекты через простой и понятный интерфейс без перегрузки.

---

## 🎨 Дизайн

- минималистичный чистый дизайн
- акцент на читаемость и контенте
- продуманная структура блоков

---

## 💻 Разработка

- дизайн сайта был разработан в Framer
- сверстан без использования фреймворков
- применял ИИ как инструмент для ускорения разработки
- HTML5
- CSS3
- JS

## ⚙️ Функционал

- кастомный слайдер проектов
- адаптивная верстка (desktop/table/mobile L)
- интерактивный sidebar для мобильного устройства
